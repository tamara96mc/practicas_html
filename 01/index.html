Secciones y lineas generales de un documento HTML5

La especificación HTML5 trae nuevos elementos a los desarrolladores web, permitiendoles describir la astructura de un documento web semántica estandarizada. Este docuemto describe estos elementos y cómo usarlos para definir el perfil de cualquier documento

Problemas resueltos por HTML5

La definición de la estructura de un documento de HTML4

  1. HTML5 quita la necesidad de elementos <div>

  2. Mezclar varios documentos es dificil: la inclusión de sub-documento en un documento principal. Esto se resuelve con HTML5 con los elementos de seccionado (<article> , <section> , <nav> , <aside> )

  3. HTML5 introduce el elemento <hgroup> que oculta todos los elementos de cabezera excepto el primero de más alto rango (por ejemplo, <hgroup><h1>Justine</hgroup></h1> )
